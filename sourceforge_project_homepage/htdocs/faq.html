Frequently Asked Questions

  1. Option xxx for tool yyy is missing

    All tools supported in this plugin have a Other options field, where you can add any needed options for your project.
    If you think that this option is of general use for other developers and should be integrated into this plugin, then make a Feature Request and we will try to integrate it.

  2. The Print Size tool quits with an error:

    ...avr-size.exe: invalid argument to --format: avr

    You have an older version of the avr-gcc toolchain. Either select a different format (sysv or berkeley) or update your avr-gcc toolchain.
