Frequently Asked Questions

  1. Option xxx for tool yyy is missing

    Most tools supported in this plugin have a Other flags or similar named fieldes, where you can add any needed flags for your project.
    If you think that your option is of general use for other developers and should be integrated into this plugin, then make a Feature Request and we will try to integrate it.

  2. The Print Size tool quits with an error:

    Update to Version 2.0.1 where this has been fixed.
