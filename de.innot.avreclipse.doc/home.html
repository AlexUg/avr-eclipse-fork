AVR Eclipse Plugin User Guide

The AVR Eclipse Plugin is an extension to the C/C++ Development Toolkit (CDT) to develop applications for the ATMEL AVR series of embedded processors.

At its core are new toolchains that know how to build the hex dump files which can be deployed on an AVR series processor. Some additional useful tools of the GNU binutils are integrated as well. Also a new viewer is included, which shows some usefull information about the selected target processor. Currently this plugin has no tools to download a program to the actual hardware (e.g. avrdude). This will be added in a future version.

The Plugin requires the GNU toolchain for the AVR Target processor. Please see the Before you begin section for installation instructions
